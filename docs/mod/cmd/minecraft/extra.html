Moderator Menu

To handle some in-game situations, there are a few options in the moderator menu:

- Spectator Mob Override: This can be used to either hide your spectator from cheaters, or change it to a vex to try to get them to trigger a cheat on a "hostile mob"
- LEMClientHelper List: Useful for knowing how to respond someone asking if they have LCH installed
- Reset Host Options: Resets the game options, useful if someone is intentionally trying to make gameplay bad
- Disable Multirounds: Removes all the remaining rounds in a game.
- Force End Round: Forces the current round to end.

/trigger moderator

Suspicious Players

Players can be marked as suspicious if they are potentially breaking the rules, marking them as suspicious will ping the @xof role on Discord when they join the server.
You can add the xof role to yourself in Discord using the /role command.

Note: This command doesn't have a [Reason] option yet, and we plan on fixing this soon.

/sus [Player]
/unsus [Player]

Chat Disabler

If things in Minecraft chat get too heated, you can temporarily disable in-game chat if needed.

/chatdisabler enablechat [true/false]
Example: /chatdisabler enablechat false

Player list

See the amount of players online on either your current server, a specific server, or all servers.

/list
List all players on your current server

/glist [Server]
You can either type the name of a specific server, or type all to see all players across all servers
