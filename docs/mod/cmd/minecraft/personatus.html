Personatus

Personatus allows you to change your Minecraft name and skin to another player's temporarily, this is useful for playing with potential cheaters to see their behavior without them knowing a mod is online.

Personatus commands require the player affected to relog for them to take effect

Note: This command is not available to trial moderators.

Return

/personatus spoof [Player] [Spoofed Name]
Changes the appearance of [Player] to the appearance of [Spoofed Name].
Important Note: The name of [Spoofed Name] MUST be a name that already exists, NameMC can be used to check if a name is used by a player.
/personatus clearSpoof [Player]
Resets the appearance of [Player] back to their real username and skin.
Note: Use the username of [Player], NOT [Spoofed Name].
/personatus checkSpoof [Player]
Checks if a player has a spoofed name and skin applied and will tell you the name they appear as
/personatus SeeDisguise [Player] [Spoofed Name]
Checks if a player in the same server as you is actually someone being disguised as someone else.
