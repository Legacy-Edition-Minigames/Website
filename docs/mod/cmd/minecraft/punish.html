Punishments

These are for issuing punishments in the Minecraft server.

These do not sync with the Discord server at the moment, we plan on having it sync in the near future.

Return

Important Note

When issuing a punishment, make sure you know the difference between both a user ban and an IP ban.

A user ban bans just the username, joining on an alt account is very easy. These should only be used for offenses that affect that account in particular, like an innapropriate skin or username.

IP bans will ban BOTH the username and IP address and are used for most offenses.

Make SURE you use the right command! For example /ban vs /ban-ip will determine what kind of ban you are issuing!

Punishment History

/banhammer list [Player]

Depending on the rule broken, you may need to check the player's punishment history to decide what to do. The punishments are listed from newest to oldest.

Bans

/ban [Player] [Reason]
/ban-ip [Player] [Reason]
/tempban [Player] [Duration] [Reason]
/tempban-ip [Player] [Duration] [Reason]
Example: /tempban-ip Steve 7d Repeated explotation of bugs to stall matches

Mutes

/mute [Player] [Reason]
/tempmute [Player] [Duration] [Reason]
Example: /tempmute Steve 7d Spamming chat even after being warned

Kicks

/kick [Player] [Reason]
Example: /kick Steve Intentionally stalling matches

Warns

Note: Warns will only appear when the player is in the server, if they disconnect before being warned they will never see it. This will be fixed in the near future.
For now, issue a lesser version of the next punishment, for example if they were spamming chat, mute them for a day instead.

/warn [Player] [Reason]
Example: /warn Steve 7d Spamming chat

Punishment Removals

Note: When a player is IP banned, you will need to unban BOTH their user and IP ban, so you will need to run /unban-ip and /unban on their username.

/unban [Player] [Reason]
/unban-ip [Player] [Reason]
/unmute [Player] [Reason]
Example: /unban-ip Steve Appealed
