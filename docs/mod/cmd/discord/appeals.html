Appeals

Commands for the appeals system on Discord

Return

Note

As a moderator, you may opt-in to participate in responding and acting on appeals sent in by players, this takes place in a separate server from the main one.

If you join the appeals server, you should make sure that you go to your Privacy Settings for the server and disable everything.

Join Appeals Server

/add-user-to-appeal [Discord User]
Add a user that is in the appeals server to the current appeal channel.

/remove-user-from-appeal [Discord User]
Remove a user from the current appeal channel.

/close-appeal
Close the current appeal channel, only run this once everything is said and done.

/appealmenu
Re-opens the create appeal menu, should only be used in #create-appeal if the button stops working.
