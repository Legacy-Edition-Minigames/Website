Minecraft Punishments via Discord

These are the commands for the Discord server for issuing punishments to the Minecraft server.

These do not sync with the Discord server at the moment, we plan on having it sync in the near future.

Return

Important Note

Note: To run commands, you will run them from Discord's slash command menu under the bot Derpbot 3.

One of the options in most commands is a Server option, the options are as follows:
menu - Menu Server
main - Battle Main
alt - Battle Alternate
exp - Battle Experimental
germany - Battle Germany

This will determine what server the command will run inside of, if you don't pick anything it will default to the Menu server.

Note for commands with a [Duration] option, you may specify none to make your punishment permanent.

Bans

/mc-ban [Player] [Server] [Duration] [Reason]
Example: /mc-ban Steve main none Cheating

Mutes

/mc-mute [Player] [Server] [Duration] [Reason]
Example: /mc-mute Steve 7d Spamming chat after a warning

Kicks

/mc-kick [Player] [Server] [Reason]
Example: /mc-kick Steve main Idling

Warns

Note: Warns will only appear when the player is in the server, if they disconnect before being warned they will never see it. This will be fixed in the near future.
For now, issue a lesser version of the next punishment, for example if they were spamming chat, mute them for a day instead.

/mc-warn [Player] [Server] [Reason]
Example: /mc-warn Steve Spamming chat

Punishment Removals

Note: When a player is IP banned, you will need to unban BOTH their user and IP ban, so you will need to run /unban-ip and /unban on their username.

/mc-unban [Player] [Server] [Reason]
/mc-unmute [Player] [Server] [Reason]
Example: /mc-unban Steve menu Appealed
