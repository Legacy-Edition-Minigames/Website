Player Reports

These are the commands for the player reports system inside of Discord

Return

VERY Important Note

Any evidence uploaded to a channel in the Player Reports category should be Downloaded and reuploaded to #report-evidence, do not copy the link to the evidence and post it there.

ALL files uploaded to a channel in Player Reports will be deleted once /close it run.

We plan on automating this process in the future, but for now all evidence must be manually saved.

/close
Closes the report, only run this once everything is resolved.

/add [Discord User]
Adds a user to the report to talk.

/remove [Discord User]
Removes a user from the report if they have been added.
