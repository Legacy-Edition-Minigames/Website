Extra Commands

These commands are uncategorized due to them serving one singular purpose and not needing too much explanation.

Return

Chat Disabler

If things in Minecraft chat get too heated, you can temporarily disable in-game chat if needed.

/mc-disablechat [server] [Muted/Unmuted]
Example: /mc-disablechat main Muted

Roles

You can add and remove select roles from users to toggle certain permissions. You can toggle the following roles:
Member - Talk perms, disables access to #verification, don't remove this just to mute someone.
Media Perms - Media upload permissions, images, videos, etc...
Soundboard Perms - Soundboard permissions.
Nickname Perms - Nickname change permissions.
TTS Perms - Permission to use the TTS bot.
Bridgeless - Removes the ability to see the bridge channels.
Bridge Muted - Disables the ability to talk in the bridge channels.
xof (Suspicious Ping) - Get pinged when a suspicious user joins the MC server.

/role [Add/Remove] [Role] [User]
Example: /role Remove Steve Soundboard Perms

User Help

These commands can be used when someone asks a common question we have a command response for, such as what the IP/Invite/Playercount is, or how to do a certain action.

/discord - Get discord invite link.
/ip - List all IP addresses.
/status - Show the playerlist of all the MC servers.
/takeall - Show a player how to use Take Everything.
