Moderation Command Guide

This is a guide on how to use commands for LEM's moderation system.

This does NOT contain the punishment guide, that is separate from this page.

Usage Note
For commands with a [Duration] option, duration can be expressed in these ways:
Seconds (15s)
Minutes (3m)
Hours (24h)
Days (7d)
Years (1y)
It can also be any combination of these (5y3d9h3m8s).
Evidence note

If you suspect someone of breaking the rules, you should do what you can to collect evidence.
You should have screen recording software set up before you begin moderating, and Replay Mod if possible.

Once you have collected evidence, post it in the #report-evidence channel on Discord. Note: Make sure to check if a post has already been made for that player before making one!
