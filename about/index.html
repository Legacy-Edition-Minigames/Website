Legacy Edition Minigames is a set of open source LCE? mini game remakes!

Currently only available for Java Edtion.

LEM focuses on remaking the Mini Games from the discontinued Legacy Console edition, while also enhancing and adding new content at the same time.

This includes quality of life improvements, modding support, re-imaginings of the game through remastered content, and new features.

LEM is developed by Emmie, Kyrptonaught and Senseiwells, for more info see the credits page.
