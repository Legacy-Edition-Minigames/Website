LEM Client Helper

LEM Client Helper is an optional enhancement mod for the server that adds some features that aren't possible on vanilla.
You do not need LCH installed to play on LEM or get most of the experience, it is just an optional enhancement.

LCH requires you to already have installed the Fabric Loader and its API, it does not support Forge currently.

Important note: Make sure you install the 1.20.4 versions of Fabric and Fabric API! LCH currently has not been ported to LEM 2.0 yet.

LEM Client Helper is incompatible with Legacy4J currently.
